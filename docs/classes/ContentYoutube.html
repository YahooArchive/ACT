ContentYoutube Class

Defined in: src/capabilities/ACT_contentYoutube.js:8
Module: ContentYoutube

The 'ContentYoutube' is a capability made to generate a 'DIV' tag use as a container for the Youtube player.

Available 'actions':

  • youtubeStart
  • youtubeStop
  • youtubePlay
  • youtubePause
  • youtubeMute
  • youtubeUnmute

Available 'triggers':

  • pause
  • play
  • complete
  • error
  • ready
  • start
  • 25percent
  • 50percent
  • 75percent

Example of 'SuperConf' use case:

 {
     id: 'youtube_player',
     type: 'content-youtube',
     classNode: 'youtube_class',
     env: ['html', 'flash'],
     css: {
         width:'352px',
         height:'198px'
     },
     youtubeConfig:{
         width: '100%',
         height: '100%',
         videoId: 'GQQMLE4FuIQ',
         suggestedQuality: 'default',
         playerVars: {
             autoplay: 1,
             controls: 1,
             color: 'white',
             disablekb: 1,
             enablejsapi: 1,
             fs: 1,
             iv_load_policy: 3,
             modestbranding: 0,
             rel: 0,
             showinfo: 0,
             loop: 0
         }
     },
     eventActions: [{
         eventType: 'start',
         actions: [
             {
                 type: 'track',
                 label: 'start'
             }
         ]
     }]
 }
