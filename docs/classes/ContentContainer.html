ContentContainer Class

Defined in: src/capabilities/ACT_contentContainer.js:326
Module: ContentContainer

The 'ContentContainer' is a capability made to generate a 'DIV' tag use as a container.

Available 'actions':

  • openURL
  • showContainer
  • hideContainer
  • changeStyles

Available 'triggers':

  • click
  • mouseenter
  • mouseleave

Example of 'SuperConf' use case:

   {
       id: 'mpu_container',
       type: 'content-container',
       classNode: 'mpu_container_class',
       env: ['html','flash','backup'],
       css: {
           width:'350'
       },
       eventConfig: [{
           eventType: 'click',
           actions: [
               {
                   type: 'openURL',
                   URLpath: 'https://www.yahoo.com',
                   URLname: 'mpu_click_clicktag_open'
               }
           ],
           timeTo: 2
       }],
       content: []
   }
