ContentImage Class

Defined in: src/capabilities/ACT_contentImage.js:8
Module: ContentImage

The 'ContentImage' is a capability made to generate a 'IMG' tag.

Example of 'SuperConf' use case:

 {
     id: 'mpu_image',
     type: 'content-image',
     classNode: 'mpu_image_class',
     env: ['html','flash','backup'],
     css: {
         width: 300,
         height: 250
     },
     imageConfig: {
         src: '',
         alt: '',
         title: ''
     }
 }
